# WBS Builder

單一離線 HTML 檔的 WBS（工作分解結構）建立工具，無需安裝或伺服器，適用於氣隙（air-gapped）生產環境。

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，無需任何設定。

```bash
open index.html      # macOS
start index.html     # Windows
```

## 功能

- **樹狀編輯**：支援最多 5 層，雙擊行內編輯節點標題
- **新增 / 刪除**：新增大項、子項；刪除前有確認告警
- **拖曳排序**：支援同層重排與跨層移動
- **節點欄位**：負責人、開始 / 結束日期、狀態（未開始 / 進行中 / 完成）、備註
- **父項狀態**：由子項狀態自動推算，不可手動更改
- **搜尋篩選**：關鍵字搜尋 + 狀態篩選，已收合子樹也參與搜尋
- **統計列**：即時顯示節點總數、最大層級、完成進度
- **WBS 模板**：新功能開發、專案開發（含 SIT / PROD 兩階段）
- **展開 / 收合**：一鍵全部展開或收合
- **匯出**：CSV（Excel 相容）、XLSX、JSON
- **匯入**：JSON 還原專案

## 技術

純 Vanilla JS + 內嵌 CSS，無任何外部依賴。XLSX 匯出使用 SheetJS，已 inline 嵌入 HTML 中維持離線可用性。
